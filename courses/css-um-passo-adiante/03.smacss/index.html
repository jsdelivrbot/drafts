  • BASE.CSS

O arquivo base.css é utilizado, dentro do SMACSS, como um arquivo que deve conter as regras básicas da aplicação. Dentro dele devem conter apenas seletores de elemento, Ex:

body, h1, h2, etc...

com variações como seletor filho ou descendente, Ex:

form > input ou body div

Em BASE.css podem ir também CSS resets.
