Flex Wrap

Utilizado para definir como o conteúdo deve se alinhar no flex container quando o container tiver uma largura ou altura fixas, o que inicialmente causará um overflow, ou seja, os flex-itens vão ultrapassar o espaço de seu flex-container.
Valores possíveis:

Wrap

faz com que os itens quebrem a linha, ou a coluna quando atingirem o limite do seu flex-container
