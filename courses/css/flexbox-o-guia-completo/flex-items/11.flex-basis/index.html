Flex Basis

Flex Basis define o tamanho inicial dos flex items que o tamanho deles seja alterado por flex-grow ou flex-shrink.

Flex Basis é a maneira correta de se definir o tamanho dos items de um container.

Ele poderá ser aplicado à largura, caso flex-direction tiver o valor: "row".
