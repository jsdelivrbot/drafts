Flex Grow

Flex Grow distribui proporcionalmente o espaço que sobra dentro de um container entre os items, por exemplo, em um container de 800px de largura, que contem três items e onde cada item recebe 100px de largura como abaixo, sobram 400px

Caso adicionemos flex-grow:1; para todos os items, eles se estenderão durante todo o espaço que falta no container. Flex-grow calcula a proporção da seguinte maneira:
Cada item recebeu flex-grow:1; temos 4 items, portanto cada item deve receber o resultado de 400px/4 (espaço em branco dividido pelo numero de partes)
