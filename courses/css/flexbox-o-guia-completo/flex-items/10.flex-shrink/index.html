Flex Shrink

Flex Shrink serve para diminuir proporcionalmente os flex-items quando existe espaço FALTANDO no container, por exemplo no container abaixo: O container tem 400px de largura;
Cada item tem 150px de largura, logo, a soma dos items estoura a largura do container (600px > 400px)

Adicionando flex-shrink:1; para cada container, o que irá acontecer:
O espaço faltante no container será calculado: 600-400 = 200px;
Como cada item tem flex-shrink: 1, cada item diminuirá na proporção de uma parte do espaõ faltante, para 4 items temos 200/4=50; então cada item PERDERÁ 50px de largura, ficando com 100px cada um
