Order

Redefine a ordem de exibição de um item independente da ordem em que está escrito no html.

Os valores possíveis são qualquer numero negativo ou positivo Os itens com order de maior valor aparecem primeiro que os de order de menor valor. Por exemplo

Sem alterar o order:

1
2
3
4

Alterando o Order:

Vamos alterar o Order do item "1" adicionando o valor "1", desta forma como todos os items tem valor padrão "0", este item aparece por último, embora esteja primeiro no html

1
2
3
4
