Flex-Direction: Row

dispõe os elementos lado a lado no main axis. Ele respeita a ordem de escrita da página se adicionarmos: html { direction: rtl; => right to left } as colunas se alinharão da direita para esquerda.

Flex-Direction: Row Reverse

Flex-Direction: Column

inverte o main axis com o cross axis. Na prática significa que o fluxo principal pelo qual o conteúdo vai se alinhar dentro do container flex vai ser na vertical ao invés da horizontal, sempre de cima para baixo.
